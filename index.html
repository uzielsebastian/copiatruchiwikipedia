HTML

HyperText Markup Language

HTML , siglas en ingles de HyperText Markup Language (lenguaje de marcado de hipertexto), hace referencia al lenguaje de marcado para la elaboracion de paginas web. Es un estandar que sirve de referencia del software que conecta con la elaboracion de paginas web en sus diferentes versiones, define una estructura basica y un codigo (denominado codigo HTML) para la definicion de contenido de una pagina web, como texto, imagenes, videos, juegos, entre otros. Es un estandar a cargo del world wile web Consortium (W3C) o Consorcio WWW, organizacion
